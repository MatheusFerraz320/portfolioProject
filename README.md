Portfólio Pessoal
Um portfólio moderno e responsivo desenvolvido com React (Vite), Tailwind CSS e animações utilizando Framer Motion. O objetivo é apresentar meus projetos, habilidades e experiência em desenvolvimento web de forma clara e interativa, com foco em desempenho e design.
Funcionalidades
- Carregamento rápido e ambiente de desenvolvimento otimizado com Vite
- Layout responsivo e mobile‑first com Tailwind CSS
- Animações suaves e transições dinâmicas com Framer Motion
- Seções dedicadas para projetos, habilidades e contato
- Interface limpa e intuitiva, com destaque para usabilidade
Tecnologias
- React.js
- Vite
- Tailwind CSS
- Framer Motion
